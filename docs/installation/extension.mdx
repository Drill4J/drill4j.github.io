To start manual testing with Drill4J, follow these guidelines:
1. Download the archive with [Google Chrome Browser Extension](https://github.com/Drill4J/browser-extension/releases/tag/v0.3.29)
2. Unzip **build** folder.
3. Click the Chrome menu icon and select Extensions from the Tools menu.
4. Enable Developer Mode in the top-right corner of the screen.
5. Click on the Load Unpacked button.
6. Navigate to **build** folder and select it.

That’s it! You can open your app page and start using the extension.
<AdditionalInformation className="space-y-2">

Don’t forget to **reload** your app page if it was opened prior to the extension installation.
</AdditionalInformation>
