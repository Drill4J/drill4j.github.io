To use Drill **Test2Code plugin** with autotests (RESTful API and Selenium-based UI [*](#notes)) 
add a special autotest agent to your test automation framework.

Supported test runners: <br/>

 ●   **junit4** <br/>
 ●   **junit5** (5.5.0+) <br/>
 ●   **testNG** <br/>

## Gradle
Add the following lines to your project **settings.gradle**:

```java
pluginManagement {
    repositories {
        maven { url 'https://oss.jfrog.org/artifactory/oss-release-local' }
    }
}
```
Add the following lines to your project **build.gradle**:
```java
plugins {
    id 'java'
    id 'com.epam.drill.agent.runner.autotest' version '0.2.2'
}
```
```java
drill {
    version = "0.15.6"
    adminHost = "localhost"
    agentId = "ExampleAgentId"
    adminPort = 8090
}
```

## Maven
Add the following lines to your project **POM**:
```java
<pluginRepositories>
    <pluginRepository>
        <id>drill</id>
        <url>https://oss.jfrog.org/artifactory/list/oss-release-local</url>
    </pluginRepository>
</pluginRepositories>
<build>
    <plugins>
        <plugin>
            <groupId>com.epam.drill.agent.runner</groupId>
            <artifactId>maven</artifactId>
            <version>0.2.2</version>
            <executions>
                <execution>
                    <goals>
                        <goal>autotest</goal>
                    </goals>
                </execution>
            </executions>
            <configuration>
                <drill>
                    <version>0.15.6</version>
                    <agentId>ExampleAgentId</agentId>
                    <adminHost>localhost</adminHost>
                    <adminPort>8090</adminPort>
                </drill>
            </configuration>
        </plugin>
    </plugins>
</build>
```
For **Surefire** plugin add args to the main properties block:
```java
<properties>
    <extraArgs>${argLine}</extraArgs> // expose to surfire if needed
</properties>
```
and add to **Surefire** plugin:
```java
<argLine>-Dfile.encoding=UTF-8 ${extraArgs}</argLine>
```
or set Drill **profile**:
```java
<profile>
    <id>drill</id>
    <activation>
        <activeByDefault>false</activeByDefault>
        <property>
            <name>env</name>
            <value>dev</value>
        </property>
    </activation>
    <pluginRepositories>
        <pluginRepository>
            <id>drill</id>
            <url>https://oss.jfrog.org/artifactory/list/oss-release-local</url>
        </pluginRepository>
    </pluginRepositories>
    <build>
        <plugins>
            <plugin>
                <groupId>com.epam.drill.agent.runner</groupId>
                <artifactId>maven</artifactId>
                <version>0.2.2</version>
                <executions>
                    <execution>
                        <goals>
                            <goal>autotest</goal>
                        </goals>
                    </execution>
                </executions>
                <configuration>
                    <drill>
                        <version>0.15.6</version>
                        <agentId>ExampleAgentId</agentId>
                        <adminHost>localhost</adminHost>
                        <adminPort>8090</adminPort>
                        <logLevel>TRACE</logLevel>
                        <logFile>drillLog.log</logFile>
                    </drill>
                </configuration>
            </plugin>
        </plugins>
    </build>
    <properties>
        <extraArgs>${argLine}</extraArgs> // expose to surfire if needed
    </properties>
</profile>
```
run it:
```cmd
-P drill
```

## Notes
<AdditionalInformation className="space-y-2">

If you use the **remote** setup for UI autotests run (Selenium GRID, SELENOID and etc.), you need to deploy proxy service:
```java
browser-proxy:
    image: drill4j/browser-proxy:0.1.0
    ports:
      - 7777:7777
```
and add additional param **browserProxyAddress** to autotests agent config:  

**Gradle**

```java
drill {
    additionalParams = ["browserProxyAddress" : "proxyHost:7777"]
    version = "0.15.6"
    adminHost = "localhost"
    agentId = "ExampleAgentId"
    adminPort = 8090
}
```

**Maven**
```java
<configuration>
    <drill>
        <additionalParams>
            <browserProxyAddress>proxyHost:7777</browserProxyAddress>
        </additionalParams>
        <version>0.15.6</version>
        <agentId>ExampleAgentId</agentId>
        <adminHost>localhost</adminHost>
        <adminPort>8090</adminPort>
    </drill>
</configuration>
```
</AdditionalInformation>
