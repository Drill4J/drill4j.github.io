# Place of Drill4J in CI/CD

Drill4J allows to make Plan - Code - Test loop in typical CI/CD process more transparent. It measures how tests affected the application.
These measurements further inform team during both Code and Test stages.

Drill4J allows to assemble the complete picture from all testing stages, rather than using disjointed coverage reports.

## Integration process

1. Pick testing stages to integrate with
2. Do a test run to supply initial data
3. Start using metrics provided
