## Testing Frameworks

<Table>

| Framework |      Supported versions     |
|:---------:|:---------------------------:|
| [NUnit](https://nunit.org)     |	      3.13.3						|
| [XUnit](https://xunit.net)     | 2.4.2, 2.5.0, 2.5.1, 2.5.2, 2.5.3	|
| [Selenium](https://www.selenium.dev)  |	4.8.1, 4.12.2, 4.14.1		|
| [Playwright](https://playwright.dev)|		  1.32.0					|
| [Specflow](https://specflow.org)|		  3.9.40						|

</Table>

- Supported versions list releases that were tested for compatibility
- Patch/Minor version updates should not break compatibility, but it is possible
- Major version change have greater chance to break compatibility
<Info>
  NOTE <br/>
  Currently, Selenium with .NET 7 is not supported 
</Info>

## Transport Layers

<Table>

| Layer		|	Supported .NET package versions |
|:---------:|:---------------------------:|

| Kafka				| 2.2.0											|
| RabbitMQ			| 6.5.0											|

</Table>

<Info>
  NOTE <br/>
  HttpClient is patched on tests launch, prior to any HTTP requests execution. Thus, any library that uses HttpClient is automatically supported. Currently, other HTTP clients such as WebClient, HttpWebRequest, RestSharper are not supported.
</Info>
