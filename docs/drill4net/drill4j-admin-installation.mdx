title: Drill4J Admin Installation
---

1. Make sure you have Docker installed.
2. Download [drill4j-admin.zip](./../../static/files/drill4net/drill4j-admin.zip).
3. Extract archive. You should have *docker-compose-admin.yml* and *.env* files.
3. Run the following command in the folder where you extracted archive.
```
docker-compose -f ./docker-compose-admin.yml up -d
```
