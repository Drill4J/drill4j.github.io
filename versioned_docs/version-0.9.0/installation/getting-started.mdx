# Getting started with Drill4J

Кратко о том, что здесь будет: запущено java приложение с агентом.

## Prerequisites

1. Нужен docker и docker-compose

## Deploying Drill4J Services

1. Скачиваем docker-compose.yml
2. Создаем .env
3. Поднимаем docker-compose up -d
4. Проверяем, что работает

## Creating API Key

1. Заходим в drill-ui
2. Заходим в раздел своих ключей
3. Генерируем ключ
4. Запоминаем/записываем

## Setup Java Agent

1. Скачиваем Java Agent zip для нужной OS
2. Заходим в drill.properties, раскомментируем свойства (drillApiUrl, drillApiKey, groupId, appId, buildVersion) и меняем значения
3. Запускаем тестируемое java приложение с -agentpath=path/to/agentlib.so -jar myapp.jar
4. Проверяем, что работает: смотрим в консоль, видим drill banner

## Metric observation

1. Открываем Metabase Builds
2. Выбираем groupId, appId, build
3. Видим coverage
