## Releases

### Release Aug 15 2025

#### New Versions

- Drill4J Backend [0.9.16](https://github.com/Drill4J/admin/pkgs/container/admin/487713489?tag=0.9.16)
- Test Agent [0.23.8](https://github.com/Drill4J/autotest-agent/releases/tag/v0.23.8)
- Application Agent [0.9.9](https://github.com/Drill4J/java-agent/releases/tag/v0.9.9)
- CI/CD Integration Plugin 0.1.10 ([Maven](https://central.sonatype.com/artifact/com.epam.drill.integration/drill-maven-plugin), [Gradle](https://central.sonatype.com/artifact/com.epam.drill.integration.cicd/com.epam.drill.integration.cicd.gradle.plugin/versions))
- Metabase Dashboards [0.9.8](https://github.com/Drill4J/drill-metabase-dashboards/releases/tag/0.9.8)

#### New Features & Improvements

- **Faster Materialized View Refresh**: Significantly reduced the refresh time for materialized views. The refresh interval is now configurable via a Quartz CRON expression, e.g., `DRILL_SCHEDULER_REFRESH_VIEWS_JOB_CRON=0 0 * * * ?` (at the start of every hour).
- **Configurable Metrics Time Window**: Introduced a metrics time window for materialized views. It can be set using the `metricsPeriodDays` property in the `/api/group-settings` API.
- **Optional Test Tracing**: Added the ability to disable test tracing to reduce unnecessary data collection, for example, in unit tests.
- **Application Archive Scanning in Maven/Gradle**: Added a command to scan application archives (.jar, .war, .ear) and send class and method metadata to Drill4J before application startup.

### Release Mar 14 2025

#### New Versions

- Drill4J Backend [0.9.11](https://github.com/Drill4J/admin/pkgs/container/admin/371078861?tag=0.9.11)
- Metabase Dashboards [0.9.3](https://github.com/Drill4J/drill-metabase-dashboards/releases/tag/0.9.3)
- Test Agent [0.23.4](https://github.com/Drill4J/autotest-agent/releases/tag/v0.23.4)
- Application Agent [0.9.7](https://github.com/Drill4J/java-agent/releases/tag/v0.9.7)
- CI/CD Integration Plugin 0.1.7 ([Maven](https://central.sonatype.com/artifact/com.epam.drill.integration/drill-maven-plugin), [Gradle](https://central.sonatype.com/artifact/com.epam.drill.integration.cicd/com.epam.drill.integration.cicd.gradle.plugin/versions))

#### New Features & Improvements  

- **Optimized Metrics Calculation**: Significantly improved metrics calculation performance with a materialized view. Metric updates are now scheduled, and users can adjust the update interval using the [`DRILL_SCHEDULER_REFRESH_VIEWS_JOB_CRON`](https://drill4j.github.io/docs/drill-services/drill-backend#scheduler-configuration) environment variable.
- **Trend Analysis for Builds**: Introduces tracking of testing and coverage changes across multiple application versions, providing deeper insights into long-term testing trends.  
- **Recommended Tests**: Enables test recommendations via a CI/CD plugin for Gradle and Maven, as well as an API endpoint. Supports precise test selection based on tags and criteria. See more in the [Test Recommendations](https://drill4j.github.io/docs/cicd-integration/cicd-integration-plugin#test-recommendations) section.  
- **Test-Centric Metrics Dashboard**: Added a dedicated dashboard offering a deeper view into test sessions, launches, and corresponding metrics from the test suite perspective.
- **Metabase Dashboards Migration Image**:
  - Changed versioning schema from `v*` to `0.9.*` to match other Drill4J components
  - Created dedicated Docker image to supply migration for Metabase-based dashboards. Users no longer need to download migration files manually - the whole process is automated.
  - __WARNING__: all customizations to previously existing dashboards will be wiped upon update, as the tables are replaced before migration is applied. All Drill4J data and metrics are preserved.
